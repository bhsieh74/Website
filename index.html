Virtual Solar System

For my final project in my Drawing on the Web class, we were assigned to develop a series of three original web-drawings. My most notable drawing was an interactive model of our solar system that was coded using HTML, CSS, and the WebGL JavaScript API.

Veterans' Affairs Database

For my high school senior year capstone project, three other classmates and I decided to collaborate to create a new and improved database of clients for the Bergen County Division of Veterans' Affairs and Services. Our database introduced new features, such as filtering clients with a search bar, exporting the list of clients to Microsoft Excel, along with adding a log-in screen to only allow certain individuals to access the confidential information. To protect the confidentiality of the veterans, the database cannot be publicly published.

Audio Visualizer

Using the knowledge I learned in my Drawing on the Web class, I created an audio visualizer of the instrumental of one of my favorite songs, "Not Springs, Love, or Cherry Blossoms" by HIGH4 ft. IU. This was created using HTML, CSS, and the WebGL JavaScript API. I also used the Three.js library and Web Audio API to gather the data from the mp3 file.
